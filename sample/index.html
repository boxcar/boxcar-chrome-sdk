Boxcar Push Notifications

To use this sample please do the following:

  1. Create a project in the Google Developer Console.
  2. Under APIs enable Google Cloud Messaging for Android.
  3. Under Credentials, create a new key and select Server key as the type.
  4. Replace "<BOXCAR_CLIENT_KEY>" in main.js with the Access Key of your Android client application from your Boxcar project.
  5. Replace "<BOXCAR_CLIENT_SECRET>" in main.js with the Secret Key of your Android client application from your Boxcar project.
  6. Replace "<YOUR PROJECT NUMBER>" in manifest.json with your own project number from the Overview page of your Google Developer Console project.
